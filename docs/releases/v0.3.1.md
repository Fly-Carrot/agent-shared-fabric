# Agent Shared Fabric v0.3.1

This small polish release makes the installed shape and startup prompt easier for new users to understand.

## Added

- `templates/layout/agent-shared-fabric.tree` showing the two-root layout:
  - `global-agent-fabric/` as the fixed governance core
  - `agent-fabric-implementation/` as the customizable implementation body
- `docs/generic-startup-snippet.md` for runtime-agnostic startup instructions.

## Changed

- Antigravity startup snippet now includes hook-first commands plus direct canonical-script fallbacks.
- Generated workspace startup snippets now include load order, dispatch order, Maestro guidance, and direct postflight fallback.
- Initializer now copies the layout template to `global-agent-fabric/LAYOUT.tree`.
