# Agent Shared Fabric v0.3.3

This release makes the template directory match the public architecture promise.

## Changed

- split `templates/` into two top-level folders:
  - `templates/governance-core/` for the fixed control plane
  - `templates/implementation-body/` for user-owned extension capability
- updated initializer paths to install governance-core templates into `global-agent-fabric/`
- added `templates/README.md` explaining the split
- added `templates/implementation-body/README.md` for custom MCP, skills, workflows, and agents
- updated README and quickstart links to the new template paths

## Verified

- smoke-tested initializer from the new template paths
- verified generated hooks return `[BOOT_OK]` and `[SYNC_OK]`
- verified no old one-level template paths remain in docs or scripts
- scanned public repo content for personal paths and obvious secret patterns
