# Agent Shared Fabric v0.2.0

This release turns the repo from an architecture sketch into a runnable local skeleton.

## Added

- one-command local initializer: `scripts/init_agent_shared_fabric.py`
- generated governance root and parallel implementation body
- runnable minimal sync scripts:
  - `preflight_check.py`
  - `sync_all.py`
  - `log_task_phase.py`
  - `postflight_sync.py`
- prompt-first and hook-based activation guide
- preflight/postflight mechanics guide
- extension body model documentation

## Clarified

- Agent Shared Fabric is the governance system.
- Fabric App is a downstream knowledge-base workstation.
- Heavy skills, MCP tools, workflows, and subagents should live in a parallel implementation folder and be referenced through registries.
