# Agent Shared Fabric v0.3.4

This release sharpens the public product story around Agent Shared Fabric as an **agent governance harness** and adds explicit structure validation documents for the two-root model.

## Changes

- added harness positioning to English and Chinese READMEs
- added `docs/structure-validation.md`
- added generated checklists:
  - `global-agent-fabric/STRUCTURE-CHECK.md`
  - `agent-fabric-implementation/STRUCTURE-CHECK.md`
- updated the initializer to copy the governance and implementation structure checks
- clarified that the governance brain is fixed while MCP, skills, workflows, and subagents remain user-owned extension body content

## Validation

- initializer smoke tests should verify both checklists are generated
- generated roots should still include hook-first boot, phase logging, and postflight sync
